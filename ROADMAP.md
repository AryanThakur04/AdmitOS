# AdmitOS Product Roadmap

> Last updated: May 2026

## Shipped (v0.1 — Demo Release)

- [x] Lead intake from CSV / API
- [x] Multi-source leads (Facebook, Website, WhatsApp referral)
- [x] Hot / Medium / Cold auto-scoring
- [x] Lead aging score
- [x] AI risk detection (no reply, missing docs, low conversion)
- [x] Lost reason analytics
- [x] Counsellor efficiency dashboard
- [x] Document completion heatmap
- [x] Referral chain tracking
- [x] Student journey replay
- [x] Conversion prediction model (heuristic + AI hook)
- [x] WhatsApp follow-up simulator
- [x] Acceptance pack generator
- [x] Alert center
- [x] Admin analytics API
- [x] Google Sheets sync stub
- [x] Demo users & sample data
- [x] Landing page + React dashboard
- [x] Docker deployment

- [ ] Facebook Lead Ads webhook integration

- [ ] Payment & deposit tracking
