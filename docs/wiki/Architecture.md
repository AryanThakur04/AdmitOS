## Backend Layers

| Layer | Path | Role |
|-------|------|------|
| Routers | `app/routers/` | HTTP endpoints |
| Services | `app/services/` | Business logic, AI, scoring |
| Models | `app/models/` | SQLAlchemy ORM |
| Schemas | `app/schemas/` | Pydantic validation |

## Key Services

- **scoring.py** вҖ” Tier, aging, conversion probability
- **risk_detector.py** вҖ” Alert generation rules
- **ai_service.py** вҖ” LLM integration with demo fallback
- **import_service.py** вҖ” CSV lead import
- **acceptance_pack.py** вҖ” PDF generation (ReportLab)

## Data Model

- `users` вҖ” Admin & counsellors
- `leads` вҖ” Core CRM entity
- `touchpoints` вҖ” Journey events
- `documents` вҖ” Per-lead doc tracking
- `alerts` вҖ” Risk notifications
- `referral_chains` вҖ” Referral graph edges
